## 🚀 How to Upload to GitHub

### Step 1 — Upload index.html
1. Go to your GitHub repo: `github.com/jeanethlapore-rgb/Jeaneth-Eguia`
2. Click on `index.html` → Click the pencil ✏️ icon (Edit)
3. Select all the old code and delete it
4. Paste the new `index.html` content
5. Scroll down → Click **Commit changes**

### Step 2 — Upload Screenshots
1. In your repo, click **Add file → Upload files**
2. Create a folder called `screenshots` by typing `screenshots/` before the filename
3. Upload all 5 Xero screenshots with these exact filenames:
   - `business-snapshot.png` (Business Snapshot report)
   - `xero-dashboard.png` (Xero Homepage/Dashboard)
   - `bank-reconciled.png` (Bank Reconciliation — Reconciled status)
   - `reconcile-screen.png` (Reconcile tab with matched transactions)
   - `account-transactions.png` (Account Transactions tab)
4. Click **Commit changes**

### Step 3 — Upload PDF Reports
1. Click **Add file → Upload files**
2. Create a folder called `reports` by typing `reports/` before the filename
3. Upload all 6 PDF reports with these exact filenames:
   - `profit-and-loss.pdf`
   - `balance-sheet.pdf`
   - `trial-balance.pdf`
   - `general-ledger.pdf`
   - `aged-receivables.pdf`
   - `aged-payables.pdf`
4. Click **Commit changes**

### Step 4 — Update Contact Details
In `index.html`, find and update:
- Your email address
- Your LinkedIn URL
